As shown in Figure 1, there are two main concepts in the ontology: Cadastral Parcel and Cadastral Property, with Cadastral Construction as a derived entity.

The class Cadastral Parcel represents the basic territorial division unit in the Spanish Cadastre. It is a geographically delimited piece of land that may contain one or more cadastral properties. This class inherits from GeoSPARQL Feature, providing geospatial capabilities. Each parcel has an associated Cadastral Reference (mandatory), an address through the callejero vocabulary, and a classification (urban/rural) defined in the external SKOS vocabulary of land classification.

The class Cadastral Property represents a real estate property registered in the Cadastre, which can be a premises, dwelling, or any independent use unit within a parcel. A cadastral property may have one or more Cadastral Constructions associated. Like the parcel, it inherits from GeoSPARQL Feature and has an associated mandatory Cadastral Reference. Additionally, it is characterized by properties such as its participation percentage in common elements, its address, its primary use (defined in the SKOS vocabulary of uses), and whether it is subject to horizontal division.

The class Cadastral Construction represents a building or construction registered on a cadastral property. It includes data on area, use, construction characteristics, and renovations. A cadastral construction may be associated with one or more cadastral properties, and inherits from GeoSPARQL Feature. It is described with properties such as its construction year, primary use, floor, staircase, door, support percentage, and horizontal division. Additionally, it may have undergone one or more renovations, represented by the Renovation class.

The class Cadastral Reference represents the complete unique reference of a cadastral entity, composed of the parcel reference (7 digits) and the property reference (alphanumeric with check digit), following the official format of the Ministry of Finance.

The class Renovation models renovation events for a construction, including the date when it occurred and the type of renovation performed. These renovations are linked to the corresponding Cadastral Construction via the has renovation relationship.
