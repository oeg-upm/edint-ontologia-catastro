As shown in Figure 1, there are three main concepts in the ontology: Cadastral Parcel, Cadastral Property and Cadastral Construction, which form a hierarchical relationship.

The class Cadastral Parcel represents the basic territorial division unit in the Spanish Cadastre. It is a geographically delimited piece of land that may contain one or more cadastral properties. It inherits from GeoSPARQL Feature, providing geospatial capabilities. Each parcel has a mandatory parcel reference (7-character alphanumeric code, e.g. 8942103VK), an address via the callejero vocabulary, and a classification (urban/rural) defined in the external SKOS vocabulary of land classification.

The class Cadastral Property represents a real estate unit registered in the Cadastre (e.g. a flat, premises, or any independent use unit within a parcel). It may contain one or more Cadastral Constructions. It also inherits from GeoSPARQL Feature and has a mandatory property reference (e.g. 0001UO). Additional properties include its participation percentage in common elements, an address, a primary use from the SKOS vocabulary of uses, and whether it is subject to horizontal division.

The class Cadastral Construction represents a building or construction unit registered on a cadastral property. It includes data on area, use, construction characteristics, and renovations. It inherits from GeoSPARQL Feature and is located on exactly one cadastral property. Properties include its construction year, primary use, floor, staircase, door, and support percentage. It may also have undergone one or more renovations, modelled by the Renovation class.

The classes Parcel Reference and Property Reference are concrete sub-classes of the abstract Cadastral Reference class. A Parcel Reference holds the 7-character alphanumeric code (e.g. 8942103VK), while a Property Reference combines the parcel reference with a property identifier and check digit (e.g. TF8006N0001HZ), following the official format of the Catastro.

The class Renovation models renovation events for a construction, including the date and type of renovation. These are linked to a Cadastral Construction via the has renovation relationship.
